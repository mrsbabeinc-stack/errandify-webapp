## 🚀 Deploy in 5 Minutes

### Local (No Internet Required)

```bash
cd /path/to/errandify
./deploy-staging.sh start
```

Access: http://localhost:5173

### Cloud (Heroku - Recommended)

```bash
heroku create errandify-staging
git push heroku main
```

Access: https://errandify-staging.herokuapp.com

### AWS EC2

```bash
docker-compose -f docker-compose.staging.yml up -d
```
