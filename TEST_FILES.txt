🐛 QUICK TROUBLESHOOTING

If something breaks, check:
  1. PostgreSQL running: brew services start postgresql@15
  2. Backend terminal shows: "Errandify API running on port 3000"
  3. Frontend terminal shows: "Local: http://localhost:5173"
  4. Browser DevTools → Network tab (check for errors)
  5. Check QUICK_TEST.md → Troubleshooting section

Can't login?
  → Check backend console for OTP log (📱 message)
  → Verify mobile matches signup (98765432)
  → OTP is valid 10 minutes only

Font size not changing?
  → Database: SELECT font_size_pref FROM users;
  → Should be 19 for age 51+
  → Clear localStorage and refresh
  → Inspect element → check font-size in HTML
