**4. Commission Logic Update** (Moderate)

```javascript
// Calculate commission based on tier
function getCommissionRate(company_id) {
  const subscription = db.subscription.findOne({ company_id });
  
  if (!subscription || subscription.status !== 'active') {
    return 0.20; // 20% for free
  }
  
  const tiers = {
    'silver': 0.18,
    'gold': 0.17,
    'platinum': 0.15
  };
  
  return tiers[subscription.tier];
}
```

**5. Monthly Ad Credit Reset** (Simple Cron Job)
